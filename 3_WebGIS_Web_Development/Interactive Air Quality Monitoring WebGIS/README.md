--------------------------------------------------------------------------------
Description:    Build a real-time air quality monitoring platform for Pakistan
                cities. Display AQI data on an interactive map with historical
                trends, health recommendations, and pollutant breakdowns.
Tools:          Leaflet.js/Mapbox, Python/Flask, AQI APIs, Chart.js
Skills Shown:   WebGIS development, API integration, interactive mapping
Difficulty:     Intermediate-Advanced
Time:           3-4 weeks
