# PROJECT 4.3: Glacial Lake Outburst Flood (GLOF) Risk Mapping
--------------------------------------------------------------------------------
Description:    Identify and map glacial lakes in northern Pakistan using
                satellite imagery. Assess GLOF risk based on lake size, dam
                stability, and downstream settlements.
Tools:          Google Earth Engine, Sentinel imagery, DEM analysis
Skills Shown:   Cryosphere studies, hazard mapping, remote sensing
Difficulty:     Advanced
Time:           3-4 weeks
