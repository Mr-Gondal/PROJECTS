# PROJECT 4.4: Smog & Air Pollution Hotspot Analysis
--------------------------------------------------------------------------------
Description:    Analyze smog patterns in Punjab using satellite aerosol data
                (AOD). Identify pollution hotspots, correlate with crop burning
                seasons, and create public health advisory maps.
Tools:          MODIS/VIIRS AOD data, Python, Google Earth Engine
Skills Shown:   Atmospheric science, health GIS, satellite data processing
Difficulty:     Intermediate
Time:           2-3 weeks
