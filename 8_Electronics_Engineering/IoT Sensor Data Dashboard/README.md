# PROJECT 8.7: IoT Sensor Data Dashboard
--------------------------------------------------------------------------------
Description:    Simulates a real-time IoT sensor network — temperature, humidity,
                air pressure, and GPS location — and streams data into a live
                monitoring dashboard with threshold alerts and trend analysis.
Tools:          Python, Streamlit, Pandas, Plotly, Folium, NumPy
Skills Shown:   IoT architecture, real-time dashboards, sensor data modeling,
                anomaly detection
Difficulty:     Intermediate
Time:           2-3 weeks
