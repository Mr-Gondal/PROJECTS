# PROJECT 8.6: Arduino / ESP32 Circuit Simulator (Wokwi)
--------------------------------------------------------------------------------
Description:    A collection of browser-simulated Arduino/ESP32 circuits built
                on the free Wokwi platform. Covers sensor reading, display
                output, PWM control, and serial communication.
Tools:          Wokwi (browser simulator), C++ (Arduino), GitHub
Skills Shown:   Embedded C++, sensor interfacing, hardware protocols (I2C, SPI,
                UART), microcontroller project structure
Difficulty:     Intermediate
Time:           2-3 weeks
