# PROJECT 8.9: Electronic Component Image Classifier
--------------------------------------------------------------------------------
Description:    A Convolutional Neural Network (CNN) trained to identify
                electronic components — resistors, capacitors, inductors, ICs,
                transistors, and LEDs — from images using transfer learning.
Tools:          Python, TensorFlow/Keras, MobileNetV2, OpenCV, Streamlit
Skills Shown:   Transfer learning, CNN fine-tuning, computer vision,
                image classification pipeline
Difficulty:     Advanced
Time:           3-4 weeks
