# PROJECT 8.3: PCB Thermal Heatmap Simulator
--------------------------------------------------------------------------------
Description:    Simulates heat distribution across a printed circuit board (PCB)
                layout using the finite difference method. Identifies thermal
                hotspots that could damage components.
Tools:          Python, NumPy, Matplotlib, Streamlit
Skills Shown:   Finite difference method, PDE solving, thermal management,
                scientific visualization
Difficulty:     Advanced
Time:           3-4 weeks
