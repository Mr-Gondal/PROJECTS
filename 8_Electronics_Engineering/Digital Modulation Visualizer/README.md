# PROJECT 8.4: Digital Modulation Visualizer
--------------------------------------------------------------------------------
Description:    Simulates and visualizes digital modulation and demodulation
                schemes — AM, FM, PSK, and QAM — complete with constellation
                diagrams and Bit Error Rate (BER) analysis under AWGN noise.
Tools:          Python, NumPy, Matplotlib, Streamlit
Skills Shown:   Digital communications, RF signal simulation, modulation theory,
                BER/SNR analysis
Difficulty:     Advanced
Time:           3-4 weeks
