# PROJECT 8.5: Satellite Link Budget Calculator
--------------------------------------------------------------------------------
Description:    Calculates the end-to-end signal budget for a satellite
                communication link — from transmitter to receiver — computing
                path loss, SNR, and link margin using the Friis equation.
Tools:          Python, NumPy, Streamlit, Plotly, ReportLab
Skills Shown:   Satellite communications, RF link analysis, antenna engineering,
                link budget analysis
Difficulty:     Intermediate
Time:           2-3 weeks
