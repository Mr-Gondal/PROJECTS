# PROJECT 8.8: Fault Detection in Power Systems (ML)
--------------------------------------------------------------------------------
Description:    A machine learning model that classifies electrical faults in
                power transmission lines using current and voltage measurements.
                Detects fault types (LG, LL, LLG, LLL) using Random Forest
                and LSTM models.
Tools:          Python, Scikit-learn, TensorFlow/Keras, Pandas, Matplotlib, Seaborn
Skills Shown:   Power systems fundamentals, supervised classification,
                time-series modeling with LSTM, ML pipeline development
Difficulty:     Advanced
Time:           3-4 weeks
